# MdNote
notebook for engineers

http://md-note.com/

## Setup(Mac OSX or Windows)

```
npm install -g typescript
npm install -g electron-prebuilt@0.31.2
npm install
```

## Debug(Mac OSX)

```
./debug.sh
```

### on Visual Studio Code

```
cmd + shift + b
```

## Packaging(Mac OSX)

```
./release.sh
```

## Debug(Windows)

```
.\debug.ps1
```

or

```
.\debug.bat
```

### on Visual Studio Code

```
ctrl + shift + b
```

## Packaging(Windows)

```
.\release.ps1
```
